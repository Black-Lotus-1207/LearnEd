About Us

What is LearnEd

LearnEd is a Student-Teacher Portal where teachers and student can sign up and teachers can add students in their class.Learn Ed contains more features like:Teachers can add or edit their student's marks after adding them in the class.Teachers can upload assignments which will be sent to all students in their class and students can download the assignments.Students can also submit their assignment but once submitted can't be changed later.Teachers can also see all the mark given by them to a student through their marks profile and can edit them if necessary.Students can see marks given to them by teachers in marks section. Student can see the list of all teachers in the portal and can message any of them. Teachers can see all the messages written by students in Inbox. User can see their profile through profile option. User can also change their password if necessary.
